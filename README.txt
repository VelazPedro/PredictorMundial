

Obtiene datos históricos de resultados mundialistas desde Wikipedia y predice el ganador de cada partido del Mundial 2026 en base a puntaje histórico acumulado, y con la intención de posteriormente juntar datos de los partidos recientes para amoldar más la predicción a la realidad.

# Requiere lo siguiente:
- Python 3.10+
- pip install pandas requests lxml beautifulsoup4
