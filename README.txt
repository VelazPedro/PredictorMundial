

Obtiene datos históricos de resultados mundialistas desde Wikipedia y predice el ganador de cada partido del Mundial 2026 en base a puntaje histórico acumulado y el resultado de los partidos desde la finalización del último mundial.
Posteriormente, espero acceder a una API del ranking fifa para también basar el puntaje conseguido por partido en la posición de la selección en el ranking y hacer de esta simulación algo más preciso.
# Requiere lo siguiente:
- Python 3.10+
- pip install pandas lxml beautifulsoup4
